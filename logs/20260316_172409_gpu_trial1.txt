Explain how memory bandwidth and PCIe transfer overhead affect large language model inference on consumer GPUs. Memory bandwidth is the rate at which data can be transferred between the GPU's memory and the CPU or other parts of the GPU. For large language models, this is a critical factor because these models require a lot of data to be loaded into GPU memory for inference. If the memory bandwidth is too low, the model may not be able to load all its weights and activations into memory quickly enough, leading to slow inference times.
